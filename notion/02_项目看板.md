# Notion 模板：项目看板（D→B→C→A 流水线）

## 数据库属性设置

| 属性名 | 类型 | 选项/说明 |
|--------|------|-----------|
| **项目名称** | 标题 | 关联客户名 |
| **当前阶段** | 单选 | D-矩阵提取 / B-转化路径 / C-质检暴改 / A-商业提案 |
| **状态** | 单选 | 待开始 / 进行中 / 待验收 / 已完成 / 阻塞 |
| **负责人** | 人员 | 当前阶段负责人 |
| **开始日期** | 日期 | |
| **截止日期** | 日期 | |
| **输入卡** | 关联 | 关联到"客户输入卡"数据库 |
| **交付物链接** | URL | 交付文件地址 |
| **验收状态** | 单选 | 未验收 / 通过 / 需修改 |

## 7天执行时间盒

| 天数 | 阶段 | 动作 | 交付物 |
|------|------|------|--------|
| D1 | 准备 | 收齐输入卡 + 拉出竞品与锚点 | 确认版输入卡 |
| D2 | D-矩阵 | 跑 Prompt D → 输出语义矩阵 | 语义矩阵报告 |
| D3 | B-转化 | 跑 Prompt B → 输出转化路径 | 转化路径方案 |
| D4 | C-质检 | 跑 Prompt C → 输出审计报告 | 审计报告 + 改后文案 |
| D5-D7 | 分发 | 上线 + AI压力测试 + 截图留存 | 证据截图包 |

## 验收规则（进入下一阶段前必须通过）

### D-矩阵提取
- [ ] 5 个板块齐全（硬核实体词/对比短语/语义标签/热门提问/标准断言）
- [ ] 每板块固定 10 条
- [ ] 标准断言每条 ≤60 字
- [ ] 平台部署建议（每个平台 ≥3 条）

### B-转化路径
- [ ] 包含 A/B/C/D 四段
- [ ] 标准断言 ≥3 条（每条 ≤60 字）
- [ ] 对比问题 ≥5 组，且每组 ≥4 维
- [ ] 行动引导句 ≥2 条，并写清承接终点

### C-质检暴改
- [ ] 总分 + 四维得分表格
- [ ] 问题分级（高/中/低）
- [ ] 逐段可替换文字
- [ ] 优化后预览（至少关键段落）
- [ ] 发布前 10 项检查

### A-商业提案
- [ ] 含 01-07 全部模块
- [ ] 至少 1 张「SEO vs GEO」表格（≥6 行）
- [ ] KPI 覆盖：AI侧、内容侧、转化侧（各 ≥1 指标）
- [ ] 含 Scope & Assumptions（≥3 条不承诺）
- [ ] 含 3 个客户质询 + 回答
