# Notion 模板：客户输入卡

## 数据库属性设置

| 属性名 | 类型 | 选项/说明 |
|--------|------|-----------|
| **客户名称** | 标题 | 必填 |
| **业务类型** | 单选 | 上游品牌 / 医美机构 / KOL团队 / 其它 |
| **核心产品** | 文本 | 主推产品/项目名称 |
| **硬核锚点** | 多选 | 注册证号 / 核心技术词 / 成分参数 / 专利号 / 论文DOI |
| **锚点详情** | 文本 | 具体的证号、参数等 |
| **竞品Top3** | 文本 | 竞品名称 + 主打词 |
| **现有资产** | URL | 官网/公众号/小红书/知乎/新闻稿链接 |
| **本次目标** | 单选 | 获客 / 招商 / 学术背书 / 终端转化 |
| **合规红线** | 文本 | 不能出现的词、不能做的对比、不能提的疗效 |
| **状态** | 单选 | 待填写 / 待审核 / 已确认 / 已启动 |
| **负责人** | 人员 | 项目负责人 |
| **创建日期** | 日期 | 自动生成 |

## 使用方法
1. 在 Notion 新建数据库，按上表添加属性
2. 点击"新建页面"，使用上述模板
3. 填写完成后，将状态改为"待审核"
4. 审核通过后，改为"已确认"，开始执行 D→B→C→A 流水线
